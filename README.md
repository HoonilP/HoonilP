## 💡 About Me

> **"글로벌 비즈니스 인사이트와 최신 기술력을 결합하여 시스템의 실질적 가치를 증명하는 엔지니어"**

🔹 **기술적 희소성** — Next.js, TypeScript 기반의 고성능 웹 서비스 구축은 물론, Vision AI와 블록체인 기술을 실무 서비스에 성공적으로 내재화한 경험을 보유하고 있습니다.

🔹 **비즈니스 임팩트** — 미얀마 마이크로파이낸스 시스템 개발 및 RAG 기반 문서 자동화 솔루션 설계 등, 복잡한 도메인 지식을 코드로 치환하여 업무 효율을 극대화하는 데 강점이 있습니다.

🔹 **실전적 실행력** — 다수의 글로벌 해커톤 참여를 통해 아이디어를 MVP 수준으로 신속히 구체화하며, 제한된 환경에서도 기술적 해결책을 제시해 성과를 창출합니다.
